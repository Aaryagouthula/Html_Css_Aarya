Program Overview

The MSIT program is designed to equip student with the skills and knowledge needed to excel in the field information technology. Our curriculum is tailored to meet the needs of both beginners and experienced professionals.

Curriculum

Data Structures and Algorithms

Learn the foundational concepts of data structures and algorithms to build efficient systems.

Database Management Systems

Understand how to design and manage databases using SQL and NoSQL technologies.

Web Development

Explore front-end and backend technologies to build dynamic, full-stack web applications.

Cloud Computing

Discover cloud platforms, virtualization, and containerization to deploy scalable applications.

Cybersecurity

Learn the essentials of protecting systens and data from malicious attacks and breaches.

Machine Learning

Gain knowledge in machine learning algorithms and how to apply them to real-world problems.

Admission Requirements

To apply for the MSIT program, you will need:

  • A Bachelor's degree in a related field
  • Official transcripts
  • Resume/CV
  • Statement of purpose
  • Two letters of recommendation
